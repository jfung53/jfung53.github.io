Training an LLM on what it means to be a man

Python Scrapy GPT-2 GQ

I made a bot to tell me what masculinity is by fine-tuning GPT-2 on 8,000 articles scraped from GQ magazine.

Images generated by masculinity Python script.

What does it mean to be a man?

 

My original idea for this project involved scraping more than just GQ magazine. I intended on including Men's Health and Men's Journal as well, but was not successful at web scraping those two. GQ, however, allowed me to scrape about 8,000 articles in one go.

 

In this project I use machine learning to examine what masculinity is.

Web Scraping

Most men's magazines have similar topics. Health, style, gear, entertainment, news, grooming, parenting, and relationships. I narrowed down the ones that appeared across more than one magazine, then I used the Python library Scrapy to crawl through the GQ website. I gathered topics, titles, dates, and the full text of each article.

Natural Language Processing

To create a dataset for training, I had to clean up the scraped articles. I removed all the sponsored content which was conveniently tagged as a topic. I used the NLP Python library SpaCy to process all of this article texts. Virgil Abloh and John Galliano are both mentioned a lot. GQ is primarily a style magazine, after all.

 

I pulled out full sentence entities only and ended up with about 6000 cleaned up text strings. This dataset lives on my Hugging Face repository.

Transformers

To create a text generation pipeline, I wrote about 40 prompts that would be the basis upon which the LLM predicts the next words. They are sentence starters like "Every man should", "Being a dad means", "A man is", "A real man", etc.

 

I used the Hugging Face Transformers framework to fine-tune GPT-2. I set it to 5 epochs and fed it my dataset of sentences. Then I fed my prompts to the text generation pipeline to get an output of over 1000 statements on what it means to be a man.

Creating a social media bot

I wrote a Python script to generate an image to post to social media. It includes a random image pulled via the Unsplash API using the search query "man in nature." I process the photo using Pillow and DrawBot, with the latter used to create an ASCII art effect to stylize the image. The generated text is styled and overlaid on top using DrawBot.

 

I had the bot running on Twitter, but decided to take it down in favour of figuring out how to get it working on Instagram. It's not currently running but will be back in action soon.

Sometimes the results are a bit strange.
